# Progress

The Progress component displays the current progress or loading status of an operation, task, or process.

## Import

```tsx
import { Progress } from '@lazy-panda-ui/lazy-panda-ui';
```

## Usage

```tsx
import React from 'react';
import { Progress } from '@lazy-panda-ui/lazy-panda-ui';

function MyComponent() {
  return (
    <Progress
      value={75}
      max={100}
      variant="determinate"
    />
  );
}
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Current progress value |
| `max` | `number` | `100` | Maximum progress value |
| `min` | `number` | `0` | Minimum progress value |
| `variant` | `'determinate' \| 'indeterminate'` | `'determinate'` | Progress variant |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Size of the progress bar |
| `color` | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error'` | `'primary'` | Color scheme |
| `radius` | `'none' \| 'sm' \| 'md' \| 'lg' \| 'full'` | `'sm'` | Border radius |
| `showValue` | `boolean` | `false` | Show progress value |
| `valueFormat` | `(value: number) => string` | `(v) => \`\${v}%\`` | Value format function |
| `animated` | `boolean` | `false` | Enable animation |
| `striped` | `boolean` | `false` | Show striped pattern |

## Loading Progress

### Auto Incrementing
```tsx
function LoadingProgress() {
  const [progress, setProgress] = useState(0);

  useEffect(() => {
    const timer = setInterval(() => {
      setProgress((prev) => 
        prev >= 100 ? 0 : prev + 10
      );
    }, 800);

    return () => clearInterval(timer);
  }, []);

  return (
    <Progress
      value={progress}
      animated
    />
  );
}
```

### Upload Progress
```tsx
function UploadProgress({ onUpload }) {
  const [progress, setProgress] = useState(0);

  const handleUpload = async () => {
    for (let i = 0; i <= 100; i += 10) {
      setProgress(i);
      await new Promise(r => setTimeout(r, 500));
    }
    onUpload();
  };

  return (
    <Stack spacing="md">
      <Progress
        value={progress}
        showValue
        color={progress === 100 ? 'success' : 'primary'}
      />
      <Button
        onClick={handleUpload}
        disabled={progress > 0 && progress < 100}
      >
        {progress === 100 ? 'Complete' : 'Upload'}
      </Button>
    </Stack>
  );
}
```

## Best Practices

1. Use appropriate variants
2. Show clear progress status
3. Maintain consistent sizing
4. Consider mobile viewports
5. Use meaningful colors
6. Provide context when needed
7. Handle edge cases

## API Reference

```tsx
  md: 6,
```

```tsx
        animation: 'progress-indeterminate 1.5s infinite',
      },
    },
  },
};
```
