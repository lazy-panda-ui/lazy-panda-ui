Backdrop

The Backdrop component renders a full-screen overlay behind content. It’s commonly used with modals, drawers, and menus.

Import​

import { Backdrop } from '@lazy-panda-ui/lazy-panda-ui';

Props​

NameTypeDefaultDescription
openboolean-Controls visibility
styleStyleProp<ViewStyle>-Container style override
childrenReact.ReactNode-Render content inside the backdrop
colorstringtheme.drawer.backdrop.colorBase color for the overlay
opacitynumbertheme.drawer.backdrop.opacityAlpha (0–1) for the overlay
onPress() => void-If provided, backdrop is pressable and triggers this callback
pointerEvents'auto' | 'none' | 'box-none' | 'box-only'-RN pointerEvents passthrough
testIDstring-Test identifier

Usage​

Basic​

<Backdrop open>
  {/* centered content by default */}
  <Text style={{ color: '#fff' }}>Loading…</Text>
  <ActivityIndicator color="#fff" />
  {/* ... */}
  {/* The layout is centered; override with style if needed. */}
</Backdrop>

Press to close​

const [open, setOpen] = React.useState(false);

<>
  <Button onPress={() => setOpen(true)}>Open</Button>
  <Backdrop open={open} onPress={() => setOpen(false)} />
</>

Custom color and opacity​

<Backdrop open color="#000000" opacity={0.7} />

With content​

<Backdrop open>
  <Card>
    <Text>Processing your request</Text>
    <Progress variant="indeterminate" />
  </Card>
  {/* You can also pass pointerEvents to control interaction routing */}
</Backdrop>

Theming​

Backdrop uses the theme’s drawer backdrop tokens:

theme.drawer.backdrop = {
  color: '#000000',
  opacity: 0.5,
};

Override via ThemeProvider if you want a different default overlay across the app.
